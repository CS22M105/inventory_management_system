# Admin Training Note: What Not To Type Into Notes

Date: July 12, 2026

Project: Katz Nursing School Inventory Management System

## Purpose

The app has Notes fields for inventory and stock activity. Notes should explain
inventory context, not personal, clinical, academic, or patient information.

## Allowed Notes

Good notes are short, operational, and inventory-related:

```text
- Restocked after simulation lab.
- Used for wound care skills practice.
- Package opened; remaining items counted.
- Moved from Room 101 to Room 202.
- Expiration checked during monthly review.
```

## Do Not Enter

Do not type these into Notes:

```text
- patient names,
- patient initials,
- diagnoses,
- real clinical details,
- grades,
- disability or accommodation details,
- Social Security numbers,
- government ID numbers,
- financial information,
- disciplinary information,
- unnecessary sensitive student information.
```

## Safer Writing Pattern

Use general lab/inventory language:

```text
Instead of: "Used for student Jane Doe's diabetes scenario."
Use:        "Used for medication administration simulation."

Instead of: "Student failed checkoff and repeated wound care."
Use:        "Used during wound care skills practice."
```

## Admin Reminder

Before exporting or sharing transaction history, quickly review Notes fields for
anything that should not have been entered. If sensitive information appears,
pause and follow the university's privacy/security process before sharing.
